# Description

This is a **Coding Tracker** program, you can add the amount of time you have spent programming and see reports about your coding habits. You can also start a **live session** that will keep running while you code, you can then save or stop it at any time. Everything gets saved on a local database. This a project for the amazing [C# Academy](https://thecsharpacademy.com/course/6/article/3/500152/False).

## Installation

1. Clone the project with `git clone https://github.com/diegopetrola/CodingTrackerCS`
2. Move the folder: `cd CodingTrackerCS\CodingTracker`
3. Run the project: `dotnet run` (requires dotnet 10).

## Features

1. Local database to save your progress
2. Live sessions to track your coding in real time
3. Coding reports with average or total on selected time periods.
